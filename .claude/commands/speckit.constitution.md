# Spec-Kit Constitution

Establish and review project principles, standards, and constraints for this project.

## Instructions

1. **Review existing documentation**:
   - Read `CLAUDE.md` for project guidelines
   - Check `package.json` for technology stack
   - Review `tsconfig.json` for TypeScript configuration

2. **Identify or establish**:
   - Core engineering principles (TDD, Tidy First, etc.)
   - Coding conventions and style guidelines
   - Architecture patterns and constraints
   - Testing requirements and standards

3. **Output**:
   - Summarize the project constitution
   - Identify any gaps or inconsistencies
   - Suggest improvements if needed

## Project Context

This is a React Native + Expo project following:

- TDD (Test-Driven Development)
- Tidy First (separate structural/behavioral changes)
- Spec-Driven Development
